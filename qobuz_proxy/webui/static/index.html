Log in to Qobuz
  1. Log in at play.qobuz.com
  2. Open DevTools (F12 or Cmd+Option+I) → Application tab
  3. Expand Local Storage → click https://play.qobuz.com
  4. Find localuser — the id field is your User ID
  5. Find your auth token (check Network tab → filter user/login → Response → user_auth_token)
